Encode Media

Notes

Encodes the given media into a video.

Syntax

encodeMedia(media: Variable, size: Type, speed: Type, customSpeed: String (Allows Variables), title: String (Allows Variables), artist: String (Allows Variables), album: String (Allows Variables), genre: String (Allows Variables), year: Integer or Variable, artwork: Variable)

Example

encodeMedia(media: Shortcut Input, size: 1920x1080, speed: Normal, customSpeed: "0", title: "Awesome Video", artist: "Zachary", album: "Videos", genre: "Indie", year: 2021, artwork: Image)
