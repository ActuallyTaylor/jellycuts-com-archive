Convert Image

Notes

Converts an image to a new format.

Syntax

convertImage(image: <#Variable#>, format: <#Type (WFImageFormat)#>, metadata: <#Boolean#>, quality: <#Double#>)

Example

convertImage(image: Clipboard, format: JPEG, metadata: true, quality: 0.433)
